[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

## Development 

1. Clone the repository
2. install dependencies 
```bash
$ npm install
```
3. Replace the .env.template for .env
4. npm run start:dev


## Running the app

```bash
# development
$ npm run start

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
